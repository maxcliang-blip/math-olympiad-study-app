# Inequalities I — Practice Problems

## Key Formulas

**AM-GM:** (a + b)/2 ≥ √(ab) for a,b ≥ 0, with equality iff a = b.
For n numbers: (a₁ + ... + aₙ)/n ≥ (a₁···aₙ)^(1/n).

**Cauchy-Schwarz:** (a₁² + ... + aₙ²)(b₁² + ... + bₙ²) ≥ (a₁b₁ + ... + aₙbₙ)²
Equality when (a₁,...,aₙ) is proportional to (b₁,...,bₙ).

**Engel Form (Titu's Lemma):** a₁²/b₁ + ... + aₙ²/bₙ ≥ (a₁ + ... + aₙ)²/(b₁ + ... + bₙ) for bᵢ > 0.

## Level 1: Direct AM-GM

1. For x > 0, find the minimum of x + 4/x.
   - ✅ AM-GM: x + 4/x ≥ 2√(x·4/x) = 4, equality at x = 2.

2. For x > 0, find the minimum of 2x + 3/x.
   - ✅ AM-GM: 2x + 3/x ≥ 2√(2x·3/x) = 2√6, equality at 2x = 3/x → x = √(3/2).

3. For positive a, b with a + b = 10, find the maximum of ab.
   - ✅ AM-GM: (a+b)/2 ≥ √(ab) → 5 ≥ √(ab) → ab ≤ 25, equality at a = b = 5.

4. For positive x, y with xy = 16, find the minimum of x + y.

5. For a, b > 0, prove a/b + b/a ≥ 2.

6. For x > 0, find the minimum of 9x² + 1/x².

## Level 3: Cauchy-Schwarz

11. Find the minimum of 1/a + 1/b given a, b > 0 and a + b = 1.

12. For positive x, y with x + 2y = 6, find the minimum of x² + y².

13. For real numbers a, b, c with a + b + c = 1, find the minimum of a² + b² + c².

14. For positive x, y, z with x + y + z = 3, find the minimum of 1/x + 1/y + 1/z.
